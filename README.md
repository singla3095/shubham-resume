# Shubham Singla — Systems That Move People

An interactive engineering resume spanning backend systems, AI, fintech, and
technical leadership across Indeed, Grab, Works Applications, and Walmart Labs.

**Live site:** [singla3095.github.io/shubham-resume](https://singla3095.github.io/shubham-resume/)

## Local development

Requires Node.js 22 or newer.

```bash
npm install
npm run dev
```

## Builds

- `npm run build` verifies the vinext production build.
- `npm run build:pages` creates the static GitHub Pages export in `out/`.

GitHub Actions publishes every push to `main` to GitHub Pages.
